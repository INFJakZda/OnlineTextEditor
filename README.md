# Online Text Editor
Editor for collaboration. 

## Task requirement:
* server only in C/C++ (C socket);
* gui for client.

## How to launch:
1. Install python3-pyqt4.
2. Launch make to build server and client.
3. Start the server.
4. Start the client.
